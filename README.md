**LOCAL INSTALLATION**

*Requirements:*
-nodejs
-npm
-Internet Browser (view the app working)

*Installation:*
In your CLI run:
npm install
npm start

*the console show an message:*
   'Server running at: http://localhost:8000' (Default app port)
